# Projeto Sprint 4 -  Analise Estatistica de Dados

## Objetivo
- Identificar o perfil dos usuário de cada plano.
- Qual investir em campanha publicitaria.

## Dataset
Projeto Bootcamp - Triple ten - Sprint 4.

## Perguntas para Analíse
- Qual plano pré-pago gera mais receita? 'Surf' ou 'Ultimate'?

## Ferramentas
- Python
- Pandas
- Matplotlib
- Seaborn
- Grafícos
- Spicpy
- Math
- Numpy

## Conclusão
O plano Ultimate é o menos menos contrato e ainda sim o mais rentável. O consumo é bem parecidas para ambos os planos. O plano Ultimate merece especial por parte do setor de publicidade.
